각 요소의 상태 값을 얻는다. - true 또는 false를 리턴한다.
prop('상태속성') - $(this).prop('readonly')

상태속성으로 사용하는 것 ; readonly, disabled, checked, selected, multiple

text객체 (readonly) :

button객체(disabled) :
확인 버튼을 클릭할 때마다 각 요소의 상태 값을 변경한다.
true -$dt; false로 false -$dt; true로 변경

현재의 상태 값을 얻어온다.
a = $('#checkTest').prop('checked');
현재 상태 값을 변경한다.
$('#checkTest').prop('checked', !a);
