속성값 가져오기
attr('name') / attr('name', 'korea') : name을 korea로 바꿔라
attr('type') / attr('type', 'password') : type을 password로 바꿔라

상태값 가져오기 - checked, selected, disabled, multiple, readonly
prop(상태값) - true 또는 false를 리턴한다.
$(this).prop('checked') . $(this).prop('checked', false) : checked를 해제하라
$('select').prop('multiple') / $(select).prop('multiple', true) : multiple로 설정하라
$(this).prop('readonly') / $(this).prop('readonly', true)
